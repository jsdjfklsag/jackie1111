# 运行指南

## 前置要求

1. **安装 Node.js**
   - 需要 Node.js 16 或更高版本
   - 下载地址：https://nodejs.org/
   - 安装完成后，打开命令行验证：
     ```bash
     node --version
     npm --version
     ```

## 运行步骤

### 方法一：使用命令行（推荐）

1. **打开命令行/终端**
   - Windows: 按 `Win + R`，输入 `cmd` 或 `powershell`
   - 或者在项目文件夹中，按住 `Shift` 右键，选择"在此处打开 PowerShell 窗口"

2. **进入项目目录**
   ```bash
   cd "e:\大三\比赛\人工智能\面向复杂环境的视光声三模态融合自主导航算法模型\AIC-2025-46644903-AC1-面向复杂环境的多模态智能导航系统\AIC-2025-46644903-AC1-面向复杂环境的多模态智能导航系统-代码与数据\AIC-2025-46644903-AC1-面向复杂环境的多模态智能导航系统-可执行程序"
   ```
   
   或者如果你已经在项目目录下，直接执行下一步。

3. **安装依赖**
   ```bash
   npm install
   ```
   
   这个过程可能需要几分钟，会下载所有必需的包。

4. **启动开发服务器**
   ```bash
   npm run dev
   ```

5. **打开浏览器**
   - 终端会显示类似这样的信息：
     ```
     VITE v5.0.8  ready in 500 ms

     ➜  Local:   http://localhost:3000/
     ➜  Network: use --host to expose
     ```
   - 在浏览器中打开 `http://localhost:3000/`
   - 界面应该会自动打开（根据vite.config.ts的配置）

### 方法二：使用 VS Code 或其他 IDE

1. **打开项目文件夹**
   - 在 VS Code 中打开整个项目文件夹

2. **打开集成终端**
   - VS Code: 按 `Ctrl + `` (反引号) 或 `终端` -> `新建终端`

3. **执行命令**
   ```bash
   npm install        # 只需第一次运行
   npm run dev     # 启动开发服务器
   ```

## 常用命令

- **启动开发服务器**: `npm run dev`
- **构建生产版本**: `npm run build`
- **预览生产版本**: `npm run preview`

## 可能遇到的问题

### 1. 端口被占用
如果 3000 端口被占用，Vite 会自动使用下一个可用端口（如 3001）。
查看终端输出的实际地址。

### 2. npm install 很慢或失败
可以尝试使用国内镜像：
```bash
npm install --registry=https://registry.npmmirror.com
```

### 3. 找不到 npm 命令
确保已正确安装 Node.js，并重启命令行窗口。

### 4. TypeScript 错误
如果遇到 TypeScript 编译错误，可以尝试：
```bash
npm install --save-dev typescript@latest
```

## 停止程序

在运行 `npm run dev` 的终端窗口中，按 `Ctrl + C` 停止开发服务器。
